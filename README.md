## Overview
  This game features an iconic famous mythical creature in the Philippines called **Manananggal**.
  <br><br>
  Our project aims to promote cultural folklore in an engaging 2D Platformer that enhances critical thinking and problem-solving skills, as well as to encourage newer generations to learn more about the rich cultural stories the Philippines has to offer. With this, the Philippines could prosper to be more competitive in game development industry and position the country for future opportunities in the global gaming market.

## Guide
### Movements
- For controlling the characters, both the **Upper** and **Lower** halves of the Manananggal uses **A**/**D** or **Left**/**Right Arrow** keys to move sideways, and the **Spacebar** to jump/flap over obstacles and platforms.
